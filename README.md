# sales-customer-anal# Sales & Customer Analysis

## 📊 Project Overview
This project analyzes sales data to understand customer behavior and business performance.

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 📈 Key Analysis
- Total Sales and Orders
- Customer Segmentation (High / Medium / Low)
- Average Order Value (AOV)
- Top Customers and Regions

## 📊 Dashboard
The project includes visualizations to show trends and insights.

## 💡 Insights
- Identified high-value customers
- Determined best-performing regions
- Analyzed sales trends over time
